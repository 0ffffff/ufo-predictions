Model Brief

About the model

This application uses a Logistic Regression model to predict the country where a UFO sighting might occur, based on duration and geographic coordinates.

The model was trained on historical UFO sighting data containing 80,000+ records from Kaggle and reached 97% accuracy on a 20,000+ sample test split.

Mission Team

About this project

UFO Predictor is a focused experiment in translating geospatial ML into a cinematic web experience.

Built with Flask and scikit-learn, the app is designed to feel exploratory while still delivering practical predictions in one interaction.
